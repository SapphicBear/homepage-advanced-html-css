# homepage-advanced-html-css
This is the final project in the Odin Project's Advanced HTML and CSS course. This webpage will be following a design doc and show a page that is fully responsive and able to serve to devices ranging from 320px to 1920 pixels wide.

Icons from [DevIcon](https://devicon.dev), and [Pictograms](https://pictogrammers.com/library/mdi/icon/share-all/).
